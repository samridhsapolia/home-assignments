# Bluetooth Home Light Controller using ESP32

## Project Description

This project controls two lights using Bluetooth communication between an Android phone and an ESP32.

Features:
- Bluetooth Control
- Two Relay Outputs
- Two Push Buttons
- Buzzer Feedback
- Bluetooth Status
- Status Command

---

## Components Used

- ESP32 Dev Board
- 2-Channel Relay Module
- 2 LEDs
- 2 Push Buttons
- Active Buzzer
- Breadboard
- Jumper Wires

---

## Bluetooth Pairing

1. Power ON ESP32.
2. Open Bluetooth Terminal App.
3. Pair with:

IIT_IoT_HomeCtrl

4. Connect.
5. Send commands.

---

## Supported Commands

| Command | Function |
|----------|----------|
| 1 | Light 1 ON |
| 2 | Light 1 OFF |
| 3 | Light 2 ON |
| 4 | Light 2 OFF |
| 5 | Both Lights ON |
| 6 | Both Lights OFF |
| ? | Show Status |
