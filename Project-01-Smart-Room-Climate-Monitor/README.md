# Project 01 - Smart Room Climate Monitor

## Objective
Monitor room temperature and humidity using ESP32 and display the data on an OLED display.

## Components Used
- ESP32 Dev Board
- DHT22 Sensor
- OLED SSD1306
- Red LED
- Green LED
- Active Buzzer

## Features
- Displays Temperature
- Displays Humidity
- OLED Output
- Status LEDs
- Buzzer Alert
- Serial Monitor Logging

## Software Used
- Arduino IDE
- Wokwi Simulator
